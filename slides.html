Dr Nic's 8 Steps

  1. Get annoyed by a defect or missing feature
  2. Find the source
  3. Checkout the source
  4. Snoop around
  5. Make changes
  6. Refresh the code from the repository (svn up)
  7. Create the patch
  8. Submit patch

http://drnicwilliams.com/2007/06/01/8-steps-for-fixing-other-peoples-code/

5 Steps

  2. Find & checkout the source
